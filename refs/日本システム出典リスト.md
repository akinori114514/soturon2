# 6.3.1 既存インフラの実績：出典リスト

## 電子政府インフラ（法人向け）

### e-Tax
- **データ**: 法人税申告 e-Tax利用率 86.2%（令和5年度）、令和4年度は9割超
- **出典**: 
  - 国税庁「e-Taxの利用状況等について」 https://www.e-tax.nta.go.jp/topics/topics_riyozyokyo.htm
  - 国税庁「令和5年度におけるオンライン（e-Tax）手続の利用状況等について」 https://www.e-tax.nta.go.jp/topics/topics_riyozyokyo/0610pressrelease.pdf

### e-Gov
- **データ**: 2023年度の総アクセス件数は約39億件
- **出典**: 内閣官房「e-Govの概要」 https://www.cas.go.jp/jp/seisaku/digital_gyozaikaikaku/kyotsu0/06kankyou.pdf

### 法人番号
- **データ**: 全法人に付番、国税庁法人番号公表サイトで公開
- **出典**: https://www.houjin-bangou.nta.go.jp/

### gBizINFO
- **データ**: 法人基本情報・行政手続き情報を集約
- **出典**: https://info.gbiz.go.jp/

### GビズID
- **データ**: 法人向け認証基盤
- **出典**: https://gbiz-id.go.jp/

### 登記情報提供サービス
- **データ**: 331円で法人登記情報をオンライン閲覧可能
- **出典**: https://www1.touki.or.jp/

## 取引インフラ

### 全銀システム・ZEDI
- **データ**: 銀行間決済インフラ、2018年全銀EDIシステム稼働
- **出典**: 全国銀行資金決済ネットワーク https://www.zengin-net.jp/

### インボイス制度
- **データ**: 
  - 適格請求書発行事業者登録件数: 約441万件（2024年2月時点）
  - 課税事業者の約97%が登録申請
  - 国税庁「適格請求書発行事業者公表サイト」で検索可能
- **出典**: 
  - 国税庁「インボイス制度について」 https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/invoice_about.htm
  - 国税庁「適格請求書発行事業者公表サイト」 https://www.invoice-kohyo.nta.go.jp/
  - 国税庁「登録件数・登録機関の推移」 https://www.nta.go.jp/taxes/shiraberu/zeimokubetsu/shohi/keigenzeiritsu/pdf/kensu_kikan.pdf

### 電子帳簿保存法
- **データ**: 2024年1月から電子取引データの電子保存が全事業者に義務化
- **出典**: 国税庁「電子帳簿保存法特設サイト」 https://www.nta.go.jp/law/joho-zeikaishaku/sonota/jirei/tokusetsu/index.htm

### DI-ZEDI（デジタルインボイス）
- **データ**: 2023年4月、全銀ネットがJP PINT対応の金融EDI情報標準を制定
- **出典**: 全国銀行資金決済ネットワーク ※要確認

## 民間信用情報インフラ

### JICC（日本信用情報機構）
- **データ**: 
  - 登録件数: 4億8,689万件（2024年5月時点）
  - 登録残高: 296兆5,023億円
  - 加盟貸金業者の個人向け貸付: 登録人数1,730.8万人、登録件数3,026.3万件

### 東京商工リサーチ
- **データ**: 企業信用調査、保有企業数 約145万社
