# 中国社会信用体系：ファクト＆用語定義

> 作成日: 2024-12-08  
> 出典: Bi Honghai (2021), 国務院綱要 (2014), bassui.md

---

## 1. 用語定義（术语定义）

### 社会信用 (Social Credit / 社会信用)
| 項目 | 内容 |
|------|------|
| **定義** | 中国の社会信用は、西洋諸国の「経済的信用（economic credit）」とは異なり、「誠実 (integrity)」「評判 (reputation)」「信頼 (trust)」に基づく概念で、市場信用を超えた広範な射程を持つ |
| **出典** | Bi Honghai p.287: "though it includes the term 'credit,' this system is not based on economic credit as understood in western countries. Rather, China's social credit is derived from integrity, reputation, and trust, the scope of which goes far beyond market credit" |
| **訳** | 「信用」という用語を含むが、西洋の経済的信用ではなく、誠実・評判・信頼に由来し、市場信用をはるかに超える範囲を持つ |

### 市場信用 vs 公共信用 (Market Credit vs Public Credit)
| 項目 | 市場信用 | 公共信用 |
|------|----------|----------|
| **主体** | 民間機関・業界団体・第三者信用サービス | 公的機関 |
| **内容** | 取引記録・信用評価情報 | 身元・資格・行政管理・司法情報 |
| **情報収集** | 本人同意が必要 | 強制的 |
| **規制方法** | 市場資源配分への影響 | 分類規制・共同褒賞/懲戒 |
| **出典** | Bi Honghai p.293 |

### 社会信用体系 (Social Credit System / SCS)
**国務院綱要 (2014) の定義:**
> 「社会主義市場経済体制と社会治理体制の重要な構成部分であり、法律・法規・標準・契約を依拠とし、社会構成員をカバーする信用記録と信用基盤インフラネットワークの健全化を基礎とし、信用情報のコンプライアンス応用と信用サービス体系を支えとし、誠信文化理念の樹立と誠信伝統美徳の発揚を内在的要求とし、守信激励と失信約束を奨懲メカニズムとし、全社会の誠信意識と信用水準の向上を目的とする」

## 2. 制度構成要素

### 2つの核心要素 (Two Key Elements)
| 要素 | 内容 |
|------|------|
| **①信用記録の保管・共有・評価** | keeping, sharing, and evaluating of credit records |
| **②インセンティブメカニズム** | various incentive mechanisms, which include rewards and punishments |
| **出典** | Bi Honghai p.1 |

### 2層構造 (Two-Level Structure)
```
第1層: 技術レベル（信用情報の収集→処理→評価）
第2層: 規制レベル（信用評価に基づく分類規制、報酬・懲戒）
```
**出典**: Bi Honghai Part II

### 4つの誠信建設領域（国務院綱要）
1. **政務誠信（政府誠信）** - 行政機関の誠信水準
2. **商務誠信（ビジネス誠信）** - 商業関係の維持・運行コスト低減
3. **社会誠信** - 社会構成員間の信頼
4. **司法公信** - 司法判決の執行可能性

## 3. 主要アクター・機関

| 機関 | 役割 |
|------|------|
| **国家発展改革委員会 (NDRC)** | 公共信用体系の調整機関 |
| **中国人民銀行 (PBOC)** | 市場信用体系の規制機関、征信業管理条例所管 |
| **最高人民法院** | 失信被執行人名簿（ブラックリスト）の管理 |

### パイロット都市
- 2015年〜: 43都市で「社会信用体系構築モデル事業」開始
- 2018年初: 12都市がモデル都市に選定
  - 杭州、南京、厦門、成都、蘇州、宿遷、恵州、温州、威海、濰坊、義烏、栄成
- **出典**: Bi Honghai p.1

## 4. 共同懲戒 (Joint Punishment / 联合惩戒)

### 典型的な制裁メニュー (p.312要約)
| # | 懲戒類型 | 例 |
|---|----------|-----|
| 1 | 公的資源配分の制限 | ナンバープレート取得制限 |
| 2 | 高額消費の制限 | 航空機・高速鉄道利用、私立校入学等 |
| 3 | 資金調達・証券での信用制限 | 融資審査 |
| 4 | 資格取得・公務員採用の制限 | 行政許可 |
| 5 | 栄誉称号の剝奪 | 表彰取消 |
| 6 | 政治的権利の制限 | 党員資格等 |

### 越境的影響リスク
> "once the joint punishment action is initiated… its effect will far exceed the local jurisdiction… a decision made by a county government may have nationwide effects"

**訳**: 共同懲戒が発動されると、その効果は地方管轄を超え、県級政府の決定でも全国的影響を及ぼし得る

**出典**: Bi Honghai p.322

## 5. 沿革・タイムライン

| 年 | 出来事 |
|----|--------|
| 1997–1999 | 人民銀行が企業・個人信用報告データベースを開始 |
| 2006 | 上記データベースがオンライン参照可能に |
| 2007 | 国務院弁公庁「社会信用体系建設に関する若干意見」 |
| 2011–2012 | 「社会信用」概念が市場信用から社会治理へ拡大 |
| 2013.3 | 征信業管理条例 施行 |
| 2013.7 | 最高人民法院「失信被執行人名簿公布規定」 |
| **2014.6** | **国務院「社会信用体系建設規划綱要（2014–2020）」公布** |
| 2015 | 芝麻信用など8社のパイロット認可（後に全社不合格） |
| 2018 | 百行征信に正式ライセンス付与 |
| 2020末 | 少なくとも35の法律・42の行政法規が社会信用条項を含む |

**出典**: Bi Honghai Part I, 国務院綱要

## 6. 「誠実」(integrity/诚信) の二面性

1. **誠実 (honesty)**: 事実を求め、欺かない
2. **約束遵守 (keeping promises)**: 約束を守る

> 「これら2つは信頼に基づく社会の基本要件であり、市場活動に限定されない」（Bi Honghai p.288）

## 7. 注意点・誤解の整理

| 誤解 | 実態 |
|------|------|
| 芝麻信用（Sesame Credit）= 中国のSCS | 芝麻信用は**市場信用**のパイロット企業の一つで、公共信用とは別系統。2018年にライセンス未取得 |
| SCS = 全自動・点数による監視システム | 現状は「Black Mirror」的な自動スコアリングではなく、**行政規制の一形態**（"old regulatory wine in a new bottle"） |
| 中央集権的一元管理 | 地方ごとに実装が異なり、データ共有は発展途上。情報アイランド問題が依然存在 |

**出典**: Bi Honghai p.284, Part I

## 8. 研究上の論点（4軸タグ）

| 軸 | 論点 |
|----|------|
| **ガバナンス** | NDRC/PBOC の役割分担、地方・中央の関係、法制化の進展 |
| **データフロー** | 信用情報プラットフォーム、Credit China、統一社会信用コード |
| **インセンティブ** | 守信激励（褒賞）と失信懲戒（懲罰）の具体的メニュー |
| **官民関係** | 市場信用（民間）と公共信用（行政）の分離と統合可能性 |
