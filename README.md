## 論文概要

**タイトル**: 中国社会信用システムの制度アーキテクチャ分析と日本への示唆
——4軸比較による企業信用制度の設計選択肢——

**研究の問い**:
- RQ1: 中国SCSの制度アーキテクチャは、4軸で見たとき公権力にどのようなパワーを与えているか
- RQ2: その構造は企業経営環境にどのような便益・コスト・リスクをもたらすか
- RQ3: 日本の企業信用制度において、便益を確保しつつリスクを抑制する設計選択肢は何か

**分析枠組み（4軸）**:
1. ガバナンス（誰がルールを定め、監督・是正できるか）
2. データフロー（信用データの収集・結合・参照の経路）
3. インセンティブ設計（優遇・制裁が誰の行動をどう変えるか）
4. 官民関係（国家・地方政府・民間の役割分担）
